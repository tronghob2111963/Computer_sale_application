================================================================================
                          TROUBLESHOOTING
================================================================================

DATABASE ERROR:
  → Run migration SQL again
  → Check PostgreSQL is running
  → Verify database exists

CORS ERROR:
  → Restart backend
  → Check CorsConfig.java exists
  → Verify frontend URL in allowedOrigins

API 404:
  → Check backend running on port 8080
  → Verify endpoint path
  → Check user ID is valid

CHATBOT NOT SHOWING:
  → Check ChatbotComponent imported in app.component.ts
  → Check <app-chatbot></app-chatbot> in app.component.html
  → Check browser console for errors

NO RESPONSE:
  → Check Gemini API key in application.yaml
  → Check backend logs
  → Verify network connection
