## Gợi ý cấu hình theo nhu cầu và ngân sách

Mỗi gói dưới đây chỉ là khởi điểm. Khi tư vấn, AI cần hỏi thêm kích thước thùng máy (ITX/mATX/ATX), nhu cầu nâng cấp sau này, số màn hình, và ưu tiên về tản/nhiệt/độ ồn. Giá mang tính tham khảo (VND).

### Văn phòng - Học tập
- **Tiết kiệm (6–8 triệu)**: CPU i3-12100 / Ryzen 5 5600G (APU), iGPU tích hợp, RAM 16GB (2x8) DDR4, SSD NVMe 500GB, PSU 400–450W 80+ Bronze, case mATX thoáng, tản stock.
- **Mượt đa nhiệm (9–12 triệu)**: CPU i5-12400 / Ryzen 5 7600, iGPU tích hợp, RAM 16–32GB DDR4/DDR5, SSD NVMe 1TB, PSU 450–500W 80+ Bronze, case mATX/ATX, tản tháp 4–6 heatpipes.

### Gaming 1080p
- **Phổ thông (14–17 triệu)**: CPU i5-12400F / Ryzen 5 5600, GPU RTX 3050 / RX 6600, RAM 16GB DDR4, SSD NVMe 1TB, PSU 550W 80+ Bronze, case ATX thoáng, tản tháp 4–6 heatpipes.
- **Cao hơn (18–22 triệu)**: CPU i5-13400F / Ryzen 5 7600, GPU RTX 3060/3060 Ti hoặc RX 6700 XT, RAM 16–32GB DDR4/DDR5, SSD NVMe 1TB, PSU 650W 80+ Bronze/Gold, tản tháp 6 heatpipes hoặc AIO 240mm.

### Gaming 1440p (2K)
- **Cân bằng (23–30 triệu)**: CPU i5-13600K / Ryzen 7 7700, GPU RTX 4060 Ti / RX 7700 XT, RAM 32GB DDR5, SSD NVMe 1TB (ưu tiên PCIe 4.0), PSU 750W 80+ Gold, AIO 240–280mm hoặc tản tháp cao.
- **Hiệu năng cao (31–38 triệu)**: CPU i7-13700K / Ryzen 7 7800X3D, GPU RTX 4070 / RX 7900 GRE, RAM 32GB DDR5, SSD NVMe 1–2TB, PSU 750–850W 80+ Gold, AIO 280mm, case ATX airflow tốt.

### Gaming 4K
- **Cao cấp (40–55 triệu)**: CPU i7-14700K / Ryzen 7 7800X3D, GPU RTX 4080 Super / RX 7900 XTX, RAM 32GB DDR5, SSD NVMe 2TB, PSU 850–1000W 80+ Gold, AIO 360mm, case ATX airflow mạnh.

### Đồ họa 2D/3D, Dựng phim, Content
- **Tối ưu ngân sách (20–28 triệu)**: CPU i5-13600K / Ryzen 7 7700, GPU RTX 3060 Ti / RTX 4060 Ti (vì NVENC/Studio), RAM 32GB DDR5, SSD NVMe 1TB, PSU 650–750W 80+ Gold, AIO 240mm hoặc tản tháp.
- **Chuyên nghiệp (35–50 triệu)**: CPU i9-13900K/14900K hoặc Ryzen 9 7900X/7950X, GPU RTX 4070/4070 Ti/4080 (ưu tiên VRAM 12GB+), RAM 64GB DDR5, SSD NVMe 2TB (chia scratch/cache và project), PSU 850–1000W 80+ Gold/Platinum, AIO 360mm, case ATX thoáng.

### Ghi chú quan trọng để AI tư vấn
- Ưu tiên RAM 2 thanh (dual-channel); với render/video nên gợi ý 32–64GB.
- Gaming 1080p/1440p: VRAM 8–12GB là “điểm ngọt”; 4K nên 12–16GB trở lên.
- PSU nên dư 20–30% so với TDP cộng gộp; chọn chuẩn ATX 3.0/3.1 và cáp 12VHPWR cho GPU đời mới.
- Ổ lưu trữ: tách ổ dự án và ổ hệ điều hành cho workload dựng phim; tối thiểu PCIe 3.0, ưu tiên 4.0 cho render/cache.
- Case phải đủ khoảng hở cho GPU (chiều dài) và radiator AIO; airflow > đèn RGB.
- Khi người dùng ưu tiên yên tĩnh: chọn quạt lớn (140mm), profile silent, và PSU semi-passive.

### Cách AI sử dụng bảng này khi tự build PC
1) Hỏi nhu cầu chính (gaming độ phân giải nào, render phần mềm gì, văn phòng?).
2) Hỏi ngân sách mục tiêu (VND) và ưu tiên: hiệu năng/êm/đẹp/nhiệt.
3) Chọn gói gần nhất theo nhu cầu + ngân sách; kiểm tra tồn kho thực tế.
4) Kiểm tra kích thước case và tương thích mainboard (ITX/mATX/ATX) trước khi chốt. 
5) Đề xuất tùy chọn nâng cấp (thêm RAM/SSD/PSU dư công suất) nếu khách còn dư ngân sách.
