   - Create new API key
   - Copy the key

2. SET ENVIRONMENT VARIABLE
   Windows CMD:
   set GEMINI_API_KEY=your-api-key-here

   $env:GEMINI_API_KEY="your-api-key-here"
   
   Linux/Mac:
   export GEMINI_API_KEY=your-api-key-here

   cd Computer-sell
   mvn clean install -DskipTests

4. RUN APPLICATION
   mvn spring-boot:run

5. TEST CHATBOT
